Data Flows

Upload: User → CloudFront → API Gateway (Cognito Auth) → Agent Lambda → S3

Processing: S3 Event → SNS Topic → SQS Queue → Ingestion Lambda (PyMuPDF extracts PDF text, Pillow extracts images) → Bedrock Vision (analyzes images) → Cohere (generates embeddings) → FAISS → S3

Infrastructure: Lambdas run in VPC private subnets. EventBridge warms up Lambdas every 5 minutes. Secrets Manager stores API keys.

Query: User → API Gateway (Cognito Auth) → Agent Lambda → Bedrock Agent → FAISS Search → Response
